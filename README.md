# Fonctionnalités HTML5 & CSS3 pour le Web Moderne

Ce projet présente les normes modernes du développement web avec les fonctionnalités HTML5 et CSS3.

## 🎯 Fonctionnalités HTML5 Introduites

### Éléments Sémantiques
- **`<header>`** - En-tête de page avec navigation
- **`<nav>`** - Menu de navigation
- **`<main>`** - Conteneur de contenu principal
- **`<section>`** - Groupement thématique du contenu
- **`<article>`** - Contenu autonome
- **`<footer>`** - Pied de page
- **`<fieldset>`** & **`<legend>`** - Éléments de formulaire groupés

### Types d'Entrée de Formulaire HTML5
- `type="email"` - Validation d'email
- `type="tel"` - Numéro de téléphone
- `type="date"` - Sélecteur de date
- `type="color"` - Sélecteur de couleur
- `type="range"` - Curseur d'entrée
- `type="text"` - Entrée texte avec attribut required

### Balises Meta
- **Charset UTF-8** - Encodage des caractères
- **Viewport** - Design réactif
- **Description** - Métadonnées SEO

## 🎨 Fonctionnalités CSS3 Introduites

### Variables CSS (Propriétés Personnalisées)
```css
:root {
  --couleur-principale: #3498db;
  --rayon-bordure: 8px;
}
```

### Techniques de Mise en Page
- **Flexbox** - Disposition unidimensionnelle avec éléments flexibles
- **CSS Grid** - Système de disposition bidimensionnel
- **Requêtes Média Réactives** - Design mobile-first

### Effets Visuels
- **Gradients** - Arrière-plans linéaires et directionnels
- **Ombres de Boîte** - Profondeur et effets d'élévation
- **Rayon de Bordure** - Coins arrondis
- **Transformations** - Rotation, mise à l'échelle, déformation
- **Transitions** - Changements de propriétés fluides

### Animations
- **@keyframes** - Animations définies
- **Propriétés d'Animation** - Durée, fonctions de minutage
- **Effets au Survol** - Pseudo-classes interactives

### Fonctionnalités Avancées
- **Pseudo-éléments CSS** (`::before`, `::after`)
- **Positionnement Sticky** - La navigation reste visible
- **Z-index & Empilement** - Contrôle des couches
- **Ombres de Texte** - Effets typographiques

## 📱 Design Réactif
- Approche mobile-first
- Dispositions de grille flexibles
- Interface conviviale pour tactile

1. Ouvrez `index.html` dans votre navigateur web
2. Explorez les éléments interactifs et les animations
